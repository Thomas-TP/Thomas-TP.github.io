# Portfolio Personnel de Thomas P. (thomastp.me)

Ce dépôt contient le code source de mon site web portfolio personnel, accessible à l'adresse [thomastp.me](http://thomastp.me/).

## 🚀 À Propos de ce Site

Ce site a été conçu pour présenter mon profil, mes compétences, mes projets et mes formations en tant qu'informaticien passionné. Il sert de vitrine numérique pour partager mon parcours et mes réalisations.

L'objectif est d'offrir une expérience utilisateur agréable, informative et moderne, reflétant mon intérêt pour les technologies web et l'innovation.

## ✨ Fonctionnalités Clés

Le site intègre plusieurs fonctionnalités pour améliorer l'expérience utilisateur et la présentation de mon profil :

*   **Design Responsive :** Entièrement adaptable aux différentes tailles d'écran (ordinateurs, tablettes, mobiles).
*   **Progressive Web App (PWA) :** Peut être "installé" sur l'écran d'accueil des appareils et offre des capacités de fonctionnement hors-ligne de base pour une meilleure accessibilité.
*   **Multilingue :** Contenu disponible en Français et en Anglais, avec un sélecteur de langue.
*   **Thème Clair / Sombre :** Possibilité de basculer entre un thème clair et un thème sombre pour le confort visuel.
*   **Optimisation pour le Partage Social :** Intégration des balises Open Graph et Twitter Cards pour un meilleur aperçu lors du partage du lien sur les réseaux sociaux.
*   **Formulaire de Contact :** Un formulaire de contact simple qui utilise un lien `mailto:` pour permettre aux visiteurs de m'envoyer un email facilement via leur client de messagerie.
*   **Chatbot Interactif :** Intégration d'un agent conversationnel Botpress pour répondre aux questions des visiteurs et fournir des informations de manière interactive (configuré séparément via Botpress Cloud).
*   **Animations Modernes :** Utilisation d'animations CSS subtiles pour dynamiser la navigation et la présentation du contenu.

## 🛠️ Technologies Utilisées

Ce site a été construit en utilisant les technologies et outils suivants :

*   **HTML5 :** Pour la structure sémantique du contenu.
*   **CSS3 :** Pour le style et la mise en page, incluant des variables CSS et des animations.
*   **Tailwind CSS :** Un framework CSS "utility-first" pour un développement rapide et une personnalisation aisée de l'interface.
*   **JavaScript (ES6+) :** Pour l'interactivité, la gestion du multilinguisme, le basculement de thème, la logique PWA (Service Worker), et la gestion du formulaire de contact.
*   **Manifeste Web (manifest.json) & Service Worker :** Pour les fonctionnalités de Progressive Web App.
*   **Font Awesome :** Pour les icônes.
*   **Google Fonts :** Pour les polices de caractères personnalisées (Poppins & Montserrat).
*   **GitHub Pages :** Pour l'hébergement du site statique.
*   **GitHub Actions :** (Potentiellement utilisé pour des automatisations futures, comme le déploiement ou des vérifications).
